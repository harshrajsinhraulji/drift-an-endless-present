# **App Name**: CyberNation

## Core Features:

- Narrative Card Display: Display decision cards with engaging narrative text and clear, swipeable choice options, mirroring the card style and information density of Lapse 2.
- Resource Management: Display dynamic resource bars (public approval, treasury, military, etc.) that visibly fluctuate in real-time based on player choices. Position and style these bars to resemble Lapse 2's resource indicators.
- Choice Impact Engine: Implement a robust system that adjusts resource levels according to the player's decisions, creating meaningful consequences, with difficulty and resource scaling similar to Lapse 2.
- Game Over Conditions: Establish clear game-over conditions when resource levels reach critical lows or unsustainable highs, with similar fail-states and end-game scenarios as Lapse 2.
- Event System: Introduce random events and long-term story arcs that add depth and replayability, akin to the event structure in Lapse 2. Events can trigger based on resource levels or past choices.

## Style Guidelines:

- Primary color: Neon Green (#39FF14) for a futuristic and vibrant feel, reflecting the advanced technology within the nation.
- Background color: Dark charcoal gray (#333333), providing a stark contrast and immersive experience.
- Accent color: Electric Purple (#BF40BF) to highlight interactive elements and key information.
- Body and headline font: 'Space Grotesk', a sans-serif font to deliver a modern and futuristic UI. If longer text is used, use 'Space Grotesk' for headlines and 'Inter' for body text.
- Use minimalist, glowing icons representing resources, choices, and game events, inspired by the icon style in Lapse 2.
- Dark themed UI with centralized cards and sidebars for the resources and progress, closely mimicking the layout and UI element placement of Lapse 2.
- Use smooth transitions and subtle glow effects to provide feedback, paying attention to the animation styles and feedback cues present in Lapse 2.
